12.5 emacspeak ¶

Emacspeak extends Emacs to be a fully functional audio desktop. This is the main emacspeak module. It actually does very little:

  • It sets up Emacs to load package-specific Emacspeak modules as each package is loaded.
  • It implements function emacspeak which loads the rest of the system.
  • emacspeak Options

12.5.1 emacspeak Options ¶

Variable: User Option emacspeak-soundscapes ¶
Whether we should turn on soundscapes on startup.

Default Value:

nil
