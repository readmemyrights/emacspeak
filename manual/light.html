13.225 light

Provide an emacs front-end to light. This is a tool that controls the brightness on laptops. To install light, sudo apt-get install light Install light setuid (sudo chmod u+s /usr/bin/light) This module is most easily used in conjunction with hydra: M-x package-install hydra
