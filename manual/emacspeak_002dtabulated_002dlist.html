12.188 emacspeak-tabulated-list

TABULATED-LIST == tabulated list mode Speech-enable tabulated lists and provide commands for intelligent spoken output

  • Emacspeak-Tabulated-List Commands

12.188.1 Emacspeak-Tabulated-List Commands

  • emacspeak-tabulated-list-next-row
  • emacspeak-tabulated-list-previous-row
  • emacspeak-tabulated-list-speak-cell

12.188.1.1 emacspeak-tabulated-list-next-row

Command: emacspeak-tabulated-list-next-row ¶
Move to next row and speak that cell

12.188.1.2 emacspeak-tabulated-list-previous-row

Command: emacspeak-tabulated-list-previous-row ¶
Move to previous row and speak that cell.

12.188.1.3 emacspeak-tabulated-list-speak-cell

Command: emacspeak-tabulated-list-speak-cell ¶
Speak current cell.
Optional interactive prefix arg speaks column header as well.
